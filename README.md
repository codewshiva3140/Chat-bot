# 🤖 Jarvis Chatbot (Gemini AI)

A simple AI chatbot built using Python and Google's Gemini API.  
This project demonstrates how to integrate a Large Language Model (LLM) into a conversational application with memory.

---

## 🚀 Features

- 💬 Real-time conversation with AI
- 🧠 Chat memory support (context-aware replies)
- ⚡ Fast responses using Gemini 2.5 Flash model
- 🛠 Beginner-friendly and easy to understand

---

## 🧰 Tech Stack

- Python
- Google Generative AI (Gemini API)
